## Run Locally

Clone the project

```bash
  git clone https://github.com/CharanRajana/Vyld-Flutter-Task.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  flutter pub get
```

Start the flutter project

```bash
  flutter run
```
