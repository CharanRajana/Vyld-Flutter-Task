## Run Locally

Clone the project

```bash
  git clone https://github.com/CharanRajana/vyld_flutter_task.git
```

Go to the project directory

```bash
  cd local-rep-location/vyld_flutter_task
```

Install dependencies

```bash
  flutter pub get
```

Start the flutter project

```bash
  flutter run
```
